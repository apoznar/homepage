Javascript ES6+

JS ES6+ - lekcja I

Kod Javascript przechowujemy w pliku main.js i odniesienie do niego wpisujemy poprzez znacznik script src="" na samym dole przed zamknięciem body. Jedna z podstawowych funkcji to alert('siema!'); - otwierające okno z wiadomością i console.log('siema!') - drukujące wiadomość w konsoli. Zmienne tworzymy formułą const a = 'Adam'; Backtick ` używany np w funkcji console.log('Zmienna ${a}'), gdzie podstawia się wcześniej stworzona zmienna.

JS ES6+ - lekcja II

Zmienne let (definiowanie kilka razy jest możliwe), const (niezmienialna) i var (już nieużywany). Typy proste - String (''), number, bigint, boolean, null, undefined, *symbol. wyrażenie typeof sprawdza typ. Nie możemy zmieniać wartości, zmieniamy połączenia między nimi a zmiennymi (niemutowalność).

DOM

Document Object Model - model dokumentu obecny na stronie. Musimy wpisać kod w JS document.querySelector('.selector') wykorzystując selektor CSS. Wydruk w konsoli pokazuje dokładnie ten element (można podać pod zmienną). Inna komenda - nazwaZmiennejQuerySelector.innerHTML = 'treść' pozwala na modyfikację elementu i nadpisanie go (to nie jest typ prosty).
