CSS

CSS - lekcja I

Konstrukcja reguł CSS wygląda w następujący sposób: selektor { właściwość: wartość; }. Możemy stworzyć w dokumencie HTML na końcu head tag "style", ale lepszą praktyką jest wstawić w to miejsce link: 'href="style.css" rel="stylesheet" type="text/css"'. Właściwości background i color dają możliwość wyboru koloru. Kolor może mieć swoją nazwę, bądź być zapisany w formacie #dddddd.

CSS - lekcja II

Możemy dodać atrybut style do znacznika, tzw. styl inline. Stylowanie jest też możliwe po #id (jako atrybut w znaczniku, tzw. selektor id, tworzymy jedno na stronie), .class (również atrybut w znaczniku; tzw. selektor klasy) i po elemencie. Możemy dodać ścieżkę np. 'nav a' będzie stylować linki w nav bez samego nav. Nie używamy !important po właściwości bo nadpisuje wszystkie zdefiniowane właściwości. Kolejność rozpatrywania stylów od najważniejszego: important, style, id, class, element. Stylowanie bliżej elementu w pliku (bądź w niższej linii) ma pierwszeństwo. Warto dodawać klasy do wszystkich elementów, ale tylko jedną do każdego rodzaju.

Życie w pudełku

Właściwość display: block sprawi, że kontener div będzie widoczny w bloku (elementy będą pod sobą), a inline sprawia że ustawiają się obok siebie. Display: inline-block pozwala dodać właściwości width i height. Możemy je zdefiniować poprzez liczbę pikseli. Właściwość max-width i max-height zmienia się wraz z responsywnością, podobnie min-width i min-height. Overflow: hidden ukrywa nam treść wystającą poza element, auto dokłada scrolla w tym przypadku. Jednostki do tworzenia stylów: 0, piksele (px), %, viewport width (vw) - % z szerokości okna a nie rodzica, viewport height (vh). Możemy określić też padding, padding-top, right, bottom, left (kolejność wpisywania rozmiaru w padding). Wartości lustrzane (możemy podać tylko 2 wartości). Podobnie ustawiamy border, style (solid, dashed, dotted), również border-color i taka jest kolejność przy samym border. Border-radius zaokrągla border. Podobnie działa margin, właściwość auto wyśrodkowuje się automatycznie - musi być podana szerokość. Box-sizing odpowiada za cały content box - właściwość border-box. Właściwość position ma własności static, relative, absolute, fixed (jest cały czas w tym samym miejscu przy scrollowaniu), sticky (rodzic nie może mieć overflowa, scrolluje się dopiero po przesunięciu ekranu). Właściwości top, right, bottom, left pomagają umiejscowić element. Z-index dzięki wysokiej wartości umiejscawia element nad innymi. Jest jeszcze float, ale nie buduj za pomocą niego layoutu (treść opływa ten element, możemy użyć własności float z kierunkiem, clear: both).

BEM - Block element modifier

Sposób nazewnictwa ułatwiający pracę np. div z klasą hand ma w sobie div z klasami hand__finger i hand__finger--thumb jako oznaczenie kciuka. W pliku CSS główna klasa najpierw, a potem jej dzieci. Używamy pełnych nazw klas, bez skrótów.
